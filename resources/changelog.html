Tru-AHK v1.8.1 Changelog

Features

  • The E key can now be customized in the dashboard.
  • A new hotkey was added (Ctrl + ,) that opens the dashboard.
  • The cursor will now move to its original location after pressing the Y key.
  • Most of the G key macros now have alternative key bindings.

Bug Fixes

  • Fixed a bug where the script would crash when pressing Shift+T.
  • Fixed a bug where if you pressed Shift+A, Shift+S, or Shift+R in an unattached Korean project manager window, it would not update it.
