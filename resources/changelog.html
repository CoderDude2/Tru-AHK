Tru-AHK v1.8.0 Changelog

Features

  • The E key can now be customized in the dashboard.
  • A new hotkey was added (Ctrl + ,) that opens the dashboard.
  • The cursor will now move to its original location after pressing the Y key.
  • Most of the G key macros now have alternative keybindings.

Bug Fixes

  • Window navigation hotkeys will now only activate on the windows associated with the active Esprit instance.
