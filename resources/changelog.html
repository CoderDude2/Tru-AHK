Tru-AHK v1.11.1 Release Notes

New Features

  • The Text-X Program can now track non-library cases with Shift + N.
  • When a case is added/removed from any list within the Text-X Program, a popup will appear in the top left corner of the screen.

Bug Fixes

  • Fixed bug where the non-library count would not change when manually adding/removing IDs.
  • Fixed bug where the popup menu for checking the connection would not save its location.
